Hallo admin baca dulu ya terkait fitur barunya

Migrate Database merupakan salah satu fitur yg berfungsi memindahkan seluruh data presensi dalam tabel ke halaman yg berbeda, alih-alih mendownload data presensi sesegera mungkin, maka sebagai developer saya menciptakan fitur ini untuk me management data presensi sesuai dengan judul dan tanggalnya, asumsinya adalah ketika anda sudah mendapatkan semua data presensi namun anda enggan untuk men download data tersebut anda bisa menggunakan fitur migrate data.

  1. Bagaimana cara menggunakan fitur ini?
  2. Klik fitur migrate data, input judul data, tanggal dan jenis data yg akan di migrate, kemudian klik submit
  3. Data tabel presensi tersebut akan dipindahkan ke halaman yg berbeda, anda dapat mengakses halaman nya pada navbar yakni Migrate Database
  4. Jika anda sudah me migrasikan data presensi ke halaman Migrate Database, anda bisa menghapus semua data presensi secara aman
  5. Kelebihan fitur ini yaitu anda bisa menghemat memori laptop/Hp anda dan anda dapat memilah data presensi mana yang akan di download/digunakan
  6. Jika terdapat error atau masalah ketika me management data, silahkan laporkan masalah pada bagian navbar Report Error

Time Picker

Sekarang admin tidak perlu melakukan input/mengetik waktu presensi secara manual lagi, fitur time picker sudah ter inisialisasi secara realtime dengan waktu saat ini admin hanya perlu menentukan jam berapa presensi telat akan ditampilkan penggunaan time picker juga sederhana
Bagaimana cara menggunakanya?

  1. Disamping button tutup, terdapat card denngan icon time pilih waktu untuk men setting presensi terlambat kapan akan di tampilkan
  2. Karena time picker bersifat realtime dengan waktu sekarang, admin hanya perlu men setting jam dan menit
  3. Setelah anda menetapkan settingan waktu klik button "Buka" untuk membuka presensi, button "Tutup" berfungsi untuk menutup presensi dengan kata lain jika presensi sudah di tutup, presensi sudah tidak menerima data lagi.
  4. Jika masih bingung terkait penggunaan fitur migrate database dan feature time picker bisa tonton video di bagian panduan ya

Sebagai developer saya berusaha mengembangkan UI dashboard yang menarik agar mudah dalam mengelola dan memanagament data, dan berusaha dalam memahami kebutuhan pengguna terkait fitur-fitur yang saya kembangkan, kedepanya saya akan mengembangkan beberapa fitur yang jauh lebih bermamfaat lagi. sekian dari saya.
